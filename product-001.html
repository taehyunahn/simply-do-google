Key Features

주요 기능

🔗

보안 URL 인보이스

각 인보이스마다 고유 액세스 토큰을 발급하여 보안 링크를 생성합니다. 관리자가 버튼 하나로 링크를 무효화할 수도 있습니다.

📊

미수금 자동 트래킹

발행된 인보이스의 입금 상태를 실시간으로 추적합니다. 미납 건수, 매출 현황을 대시보드에서 한눈에 파악할 수 있습니다.

📄

PDF 저장 & 이메일 발송

인보이스를 깔끔한 PDF로 저장하고, 고객에게 HTML 형식의 안내 메일을 자동 발송합니다.
